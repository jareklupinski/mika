Puck Setup View Log
Wi-Fi Network Name
This Puck needs to connect to the internet to check your Imgur account for reputation changes. Please choose an Access Point that can connect to the internet.

Wi-Fi Password
If your Access Point needs a password or passphrase to connect, type that in here.
My Name
After Setup is complete, you can reconnect to this Puck at any time by connecting to the Access Point this Puck will create. The newly created Access Point will be called the name you type in here.
My Password
The Access Point this Puck creates will be protected using this password. Make sure you remember it! If you ever forget your password, you will have to press the Reset button and reset the Puck to its factory settings, then run Setup again.
Imgur Account
This Puck is configured to connect to Imgur and monitor the Reputation of any user. Please type the Imgur username of the user you wish to monitor here. If the Puck cannot retrieve the user's name, the middle dot will blink red and yellow. Please doublecheck the user name matching their name on Imgur exactly.
Save Changes
Press Save to save the settings above to the Puck. The Puck will restart and create an Access Point. The Access Point's name will be the name typed in above. Connect to this newly created Access Point and point your browser to http://1.1.1.1 to access the Puck's dashboard and manage settings. If you need to restore the Puck to factory settings, press the Reset button on the Puck.
